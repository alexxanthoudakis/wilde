Data Export

All exports are UTF-8 CSV files. The first row contains header information. Fields are quoted and separated by commas.

Volume of news
Export all of the metadata for all the reports as
  1. ID
  2. Date
  3. Word count
  4. Document match count
  5. Paragraph match count
  6. Newspaper title
  7. Region
  8. Status
Matching documents
Export all of the document-level matching data as
  1. Source ID
  2. Source date
  3. Source paper
  4. Source region
  5. Target Id
  6. Target date
  7. Target paper
  8. Target region
  9. Match percentage
